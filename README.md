Fuse Youtube bindings
=====================

Library to use [Youtube](https://www.youtube.com/) in [Fuse](http://www.fusetools.com/).

## Installation

Using [fusepm](https://github.com/bolav/fusepm)

    $ fusepm install https://github.com/bolav/fuse-youtube
